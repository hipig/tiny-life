# Pixel Spaces Asset Drop

This folder is reserved for Netherzapdos Pixel Spaces assets:
https://netherzapdos.itch.io/pixel-spaces

Suggested placement:

- `building/`: apartment shell, lobby pieces, construction signs, facade trims.
- `tileset/`: wall, floor, ground, and room interior tilesets.
- `furniture/`: furniture atlases or individual furniture sprites.
- `npc/`: tenant spritesheets or per-character frame folders.
- `ui/`: panel, button, and decorative UI textures.
- `icons/`: compact icons for needs, categories, rent, coins, and tasks.
- `effects/`: bubbles, highlights, floating coins, and small VFX.

Runtime configs can reference assets through `asset` dictionaries. Supported types are:

- `placeholder`
- `single_sprite`
- `atlas_region`
- `spritesheet_frame`
- `spritesheet_animation`
